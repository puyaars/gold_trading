## Get started

### Instalation
In order to run the server, you'll need a recent version of nodejs
you also need a mysql database and redis database


### Configuration
create `.env` file in project root with items below:

```
DATABASE_URL=
REDIS_URL=
MINIO_URL="
MINIO_PORT=
MINIO_USER=
MINIO_PASS=
```
